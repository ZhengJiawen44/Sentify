YT Sentiment Analyzer Web App

A tool for analyzing YouTube video comments and their sentiment.

Overview

The YT Sentiment Analyzer Web App allows you to analyze the sentiment of comments on YouTube videos. It utilizes the YT_Sentiment Python package to fetch and process comments, categorizing them as positive, negative, or neutral, and providing sentiment scores for further analysis.

Features

  • Fetch Comments: Retrieve up to 12,000 comments from a specified YouTube video.
  • Sentiment Analysis: Classify comments into positive, negative, or neutral categories using VADER sentiment analysis.
  • Comprehensive Reporting: Display categorized comments and sentiment polarity scores.

How It Works

  1. Fetch Comments: The app uses the YouTube Data API to collect comments from a given video URL. Hyperlinks, HTML tags, and excessive emojis are filtered out.
  2. Analyze Sentiment: VADER sentiment analysis evaluates each comment, categorizing them as positive, negative, or neutral based on their content.
  3. Display Results: The app displays categorized comments and sentiment scores in an easy-to-read format.

API Usage

Initialization

from YT_Sentiments.sentiment import analyzer

# Replace with your API key and video URL
obj = analyzer("YOUR_API_KEY", "https://www.youtube.com/watch?v=VIDEO_ID", 100)

Methods

  • get_analysis(): Returns a comprehensive analysis of comments.
  • get_positive(): Retrieves a list of positive comments.
  • get_negative(): Retrieves a list of negative comments.
  • get_neutral(): Retrieves a list of neutral comments.
  • get_polarity(): Retrieves sentiment polarity scores for each comment.

Helper Functions

  • getSentimentScore(comment): Analyzes the sentiment of a single comment.
  • getComments(LINK, API_KEY, LIMIT): Fetches and cleans comments from a YouTube video.
